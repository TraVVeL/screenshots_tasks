register code - описание для кода
2. 
Воспользовался готовым классом Authentification.
Написал код для обработчика btnRegister_Click, проверяющий ведённые пользователем данные, где ввод данных не должен быть пустой строкой

Написал функцию checkAccount, которая подключается к базе данных и делает запрос на изьятии все столбцов со значением username, если
пользователь существует, выводим окно с категорией "Error".
Иначе, если пользователя не существует и веденный данные правильны вызов функции insertData.
InsertData создаёт запрос и соедиенение с базой данных, как только запрос будет выполнен, вывод диалогового окна с категорией "Success".
